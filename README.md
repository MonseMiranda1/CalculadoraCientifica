# 🧮 Calculadora Científica en Java

Proyecto desarrollado en Java que implementa una calculadora científica por consola, incorporando operaciones básicas y avanzadas, junto con mejoras de validación y almacenamiento de historial.

---

## 📌 Características

✔️ Operaciones básicas:

* Suma
* Resta
* Multiplicación
* División

✔️ Operaciones avanzadas:

* Potencia
* Raíz cuadrada
* Raíz cúbica
* Módulo (residuo)
* Porcentaje

✔️ Funcionalidades extra:

* Cálculo de área de un círculo (uso de `Math.PI`)
* Historial de operaciones
* Validación de entrada con `Scanner.hasNextDouble()`
* Menú interactivo en consola

## 🧠 Estructura del programa

El programa utiliza:

* `Scanner` → para entrada de datos
* `ArrayList` → para guardar historial
* `Math` → para operaciones matemáticas avanzadas

Además, incluye un método personalizado:

```java
leerNumero()
```

Que permite validar correctamente los datos ingresados por el usuario.
